Modules
🕵️
Scraper
scraper.py
Polls configured subreddits every 5 minutes. Detects posts and comments containing 1031 exchange keywords. Tracks already-seen IDs to avoid duplicates.
PRAW SQLite Polling loop
🤖
Responder
responder.py
Sends the original post to Claude with a custom system prompt. Generates a helpful, non-promotional reply tailored to the user's specific question.
Claude API claude-sonnet-4 System prompt
📧
Emailer
emailer.py
Sends an HTML email with the original post, Claude's draft, and one-click Approve / Reject buttons. Uses Gmail SMTP with App Passwords.
Gmail SMTP HTML email App Password
🖥️
Approval Server
approval_server.py
Lightweight Flask server that receives approve/reject clicks from the email. Updates post status in the database and triggers the poster automatically.
Flask Webhook localhost:5055
📤
Poster
poster.py
Posts the approved reply as a Reddit comment or reply using the official Reddit API. Handles both submissions and comment threads.
PRAW Reddit API OAuth2
🗄️
Database
database.py
SQLite persistence layer. Tracks every detected post, its status through the pipeline, the generated draft, and deduplication of seen IDs.
SQLite State machine Deduplication
